# Tattoo
This is the code I'm using to generate my tattoo
for my right thigh. I want to show a 1D line plot
where the vertical portions represent the probability
distribution of a prior, observed, and posterior
model.

# Meaning
This is a public repo, and I think the intricacies
of my decision are something I would only share
with the people I know if they had enough interest
to ask. 

But the basis of this tattoo is that I 
want to mark myself with the idea of incorporating
new knowledge, being honest and addressing my own
uncertainties.

Also, as a fun side thing, my thesis has to do 
with Bayes' Theorem, so this is serving as a fun
exercise for trying to intimitely understanding 
how this theory is applied, and having a close
connection with my own tattoo.
